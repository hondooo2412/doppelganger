# Doppelganger - 法務チェックリスト

## 掲示板フェーズ（先行リリース可能）

### ✅ 対応済み（実装レベル）
- [x] 禁止ワードフィルター（電話番号、メアド、LINE ID、SNSアカウント等）
- [x] 通報機能（5種類の通報理由 + 自由記述）
- [x] 出会い目的の投稿をフィルタリング
- [x] URL投稿の禁止
- [x] 匿名ID（実名不要）
- [x] RLS（自分のファミリー/タイプの板のみアクセス可能）
- [x] BAN機能（ban_status + violation_count）

### 🔲 リリース前に必要な対応
- [ ] **プライバシーポリシー** の作成と掲載
  - 収集する個人情報の種類（メールアドレス、診断結果）
  - 利用目的
  - 第三者提供の有無
  - データ保持期間
  - ユーザーの権利（削除請求等）
- [ ] **利用規約** の作成と掲載
  - 禁止事項（個人情報の投稿、出会い目的、誹謗中傷等）
  - 投稿の削除基準
  - アカウント停止の基準
  - 免責事項
  - 準拠法（日本法）
- [ ] **同意取得フロー** の実装
  - 新規登録時にプライバシーポリシー・利用規約への同意チェックボックス
- [ ] **プロバイダ責任制限法** への対応
  - 権利侵害の申し出に対する対応フロー
  - 発信者情報開示請求への対応方針

### 📌 参考: 電気通信事業法
- 掲示板のみ（小規模）の場合: **届出不要**
- DM機能追加時: 要再確認（電気通信事業に該当する可能性）

## マッチングフェーズ（届出完了後に解放）

### 🔲 リリース前に必須の対応

#### 出会い系サイト規制法
- [ ] **公安委員会への届出**
  - 事業開始の前日までに届出
  - 届出先: 事業所所在地の都道府県公安委員会
  - 届出書類: インターネット異性紹介事業届出書
- [ ] **年齢確認機能** の実装
  - 身分証（運転免許証、マイナンバーカード等）の画像アップロード
  - 生年月日の確認（18歳以上であること）
  - 確認完了までマッチング機能は利用不可
- [ ] 児童（18歳未満）の利用禁止の明記

#### 特定商取引法
- [ ] **特定商取引法に基づく表記** ページの作成
  - 事業者の名称
  - 代表者名
  - 所在地
  - 連絡先（メールアドレス、電話番号）
  - 販売価格（マッチング1回500円）
  - 支払方法（アプリ内課金）
  - 返金ポリシー
  - サービスの提供時期

#### 決済関連
- [ ] Apple Developer Program 登録（年間$99）
- [ ] Google Play Developer 登録（$25 一回限り）
- [ ] アプリ内課金（IAP）の実装
- [ ] Apple/Googleの審査ガイドラインへの準拠
